\doxysubsection{Detailed Description}
Class that works as a token supplier for the \mbox{\hyperlink{classemthp_1_1Parser}{Parser}} class. 

The class consists of two public functions, one for getting the next \mbox{\hyperlink{structemthp_1_1Token}{Token}} and the other for going back one position.
