\doxysubsection{Detailed Description}
This is the main header file for the emath library. 

This file declares the functions of the emath library.~\newline
It consists of two classes, the Monomial class and the Polynomial class.~\newline
The Monomial class declares all the functions, operators, and properties for representing and operating monomials.~\newline
The Polynomial class declares all the functions, operators, and properties for representing and operating polynomials.~\newline
